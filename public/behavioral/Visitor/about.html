Посетитель

Посетитель — это поведенческий паттерн проектирования, который позволяет добавлять в программу новые операции, не изменяя классы объектов, над которыми эти операции могут выполняться.

Посетитель — это поведенческий паттерн, который позволяет добавить новую операцию для целой иерархии классов, не изменяя код этих классов.

Результат выполнения

Клиентский код работает со всеми посетителями через базовый интерфейс Посетителя:

A + ConcreteVisitor1

B + ConcreteVisitor1


Это позволяет одному и тому же клиентскому коду работать с разными типами посетителей:

A + ConcreteVisitor2

B + ConcreteVisitor2
